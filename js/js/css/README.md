# Digital Memory Capsule Platform

A web-based application that allows users to securely store personal memories
and unlock them at a chosen future date for reflection and personal growth.

## Features
- Add personal memories
- Set future unlock dates
- View unlocked memories
- Simple and user-friendly UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Local Storage (can be extended to database)

## Future Enhancements
- User authentication
- Cloud database integration
- Encryption for memory security
